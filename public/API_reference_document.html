Product

Provides CRUD operation on "/product" endpoint

  1. GET

    Retrieve information about all products

    Sample Endpoint

    • GET /product

    Response Information

    Product Informations (HTTP response 200)

    All the informations about products in JSON format

    Sample Response

    • [ { "id": 1, "image": "https://tinyurl.com/yc3fbz85", "brand": "Apple", "model": "iphone5", "os": "Android", "screensize": 5 }, { "id": 2, "image": "https://tinyurl.com/yc3fbz85", "brand": "Apple", "model": "iphone5", "os": "iOS", "screensize": 5 } ]

    Error codes

    500
    Internal Server Error

  2. GET

    Retrieve information about a single products by id

    Sample Endpoint

    • GET /product/:id

    URL parameters

    Product Id Integer

    Required: Id number of the product.

    Not null. Not negative.

    Response Information

    Product Informations (HTTP response 200)

    All the informations about the product with given ":id" in JSON format

    Sample Response

    • { "image": "https://tinyurl.com/yc3fbz85", "brand": "Apple", "model": "iphone 5", "os": "iOS", "screensize": 8 }

    Error codes

    400
    Bad Request
    404
    Product with given ":id" not found

  3. POST

    Creates a new product in database

    Sample Endpoint

    • POST /product

    Body values

    image url

    URL address of the product's picture.

    Not Null

    brand string

    Brand name of the product.

    Not Null

    model string

    Model name of the product.

    Not Null

    os string

    Name of operating system of the phone.

    Not Null

    screensize number

    screensize of the phone as integer

    Not Null / Positive Integer

    Response Information

    Product Id (HTTP response 201)

    JSON data tells that the data has been created with unique id number

    sample request

    • { "image": "https://tinyurl.com/yc3fbz85", "brand": "Xiaomi", "model": "iphone5", "os": "Android", "screensize": 5 }

    Sample Response

    • "The item has been created with the id of 3"

    Error codes

    500
    Internal Server Error.

  4. PATCH

    Updataes the partial or whole information of the item with given ":id"

    Sample Endpoint

    • PATCH /product/:id

    URL parameters

    Product Id Integer

    Required: Id number of the product.

    Not null. Not negative.

    Body values

    image url

    URL address of the product's picture.

    brand string

    Brand name of the product.

    model string

    Model name of the product.

    os string

    Name of operating system of the phone.

    screensize

    screensize of the phone as integer

    Positive Integer

    Response Information

    No Informations (HTTP response 204)

    All the informations about the product with given ":id" in JSON format

    sample request

    • { "image": "https://tinyurl.com/yc3fbz85", "brand": "Xiaomi", "model": "iphone5", "os": "Android", "screensize": 5 }

    Sample Response

    Error codes

    400
    Bad Request
    404
    Product with given ":id" not found

Delete

Provides CRUD operation on "/delete" endpoint

  1. DELETE

    Deletes all the information about products exist in database.

    Sample Endpoint

    • DELETE /delete

    Response Information

    No Informations (HTTP response 204)

    All the information of the products have been deleted

    Sample Response

    Error codes

    500
    Internal Server Error.

  2. DELETE

    Deletes all the information of the item with given ":id"

    Sample Endpoint

    • DELETE /delete/:id

    URL parameters

    Product Id Integer

    Required: Id number of the product.

    Not null. Not negative.

    Response Information

    No Informations (HTTP response 204)

    Product information with given ":id" has been deleted.

    Sample Response

    Error codes

    400
    Bad Request
    404
    Product with given ":id" not found
